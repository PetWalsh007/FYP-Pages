A modular, containerised platform for real-time, adaptive data analysis in industrial environments.

View Objectives

Core Highlights

  • ⚙️ Deployed in a modular, containerised environment using Proxmox VE and LXC
  • 🔄 Python-based abstraction layer supports ODBC and REST access to multiple external databases or data stores
  • 🧠 Backend supports adaptive time-series analysis using Dynamic Time Warping (DTW) and various other statistical processing and basic analysis methods, either system defined or user defined
  • 🚀 Redis in-memory caching enables fast query reuse and reduced database load
  • 🛡️ PostgreSQL internal DB used for configuration storage and detailed audit logging, aligned with 21 CFR Part 11 & EU Annex 11
  • 📈 Dash-powered front-end for interactive querying, data visualisation, and analysis execution. Client side storage to use browser cache for session specific information
  • 🧩 Custom startup and shutdown routines at container and script level, plus modular configurations and logging for each service
